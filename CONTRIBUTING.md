# Git 提交规范

```
类型(scope): 描述

可选: 详细说明
```

## 类型

- `day`: 每日学习进度（例: day03: 完成环境图 + Hog checkpoint）
- `note`: 笔记更新（例: note: 添加递归章节笔记）
- `lab`: 实验完成（例: lab03: 完成递归Lab）
- `hw`: 作业完成（例: hw02: 完成高阶函数作业）
- `project`: 项目进度（例: project(hog): 完成Hog第一阶段）
- `fix`: 修复bug
- `docs`: 文档更新
- `chore`: 杂项

## 示例

```
day05: 完成Sequences + Data Abstraction + Lab04

- 看完Lec 10-12
- 完成Lab 04 (Tree Recursion)
- 笔记已同步到Notion
```
